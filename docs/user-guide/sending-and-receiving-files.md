# Sending & Receiving Files

## How to Send a File (Images, Videos, Audio, Documents)

Share files such as images, videos, music, screenshots, and documents directly through a **Liberdus chat**.

### Steps: Sending a File

1. Open the **chat** with the person you want to send files to.  
2. Tap the **+** next to the message input bar.  
3. Select **“Send a file.”**  
4. Choose a file from your device (**image**, **video**, **audio**, or **document**).  
5. To attach more files, tap the **+** and repeat the process.  
   - You can attach multiple files but must add them **one at a time**.  
6. Each file will appear in the **preview area** before sending.  
7. When all files are attached, tap the **Send**.  
8. Wait a few seconds while the files upload and send through the network.  

All selected files will appear in the chat, ready for the recipient to **view or download**.

### File Rules & Limitations

| Rule | Details |
|------|----------|
| **Max file size:** 10 MB per file | Each individual file must be under 10MB. |
| **Attach multiple files** | Add them one-by-one, then send all together. |
| **Supported formats** | Images, videos, audio files, and common document types. |
| **Tolls still apply** | If the recipient has a toll, it must be paid before sending. |
| **Secure delivery** | Files are encrypted and follow the same privacy model as messages. |

### Important

- You can send **as many files as you want**, as long as each one is under 10MB.  
- The recipient can tap any file to **view, play, or download** it.

## How to Save or Download a Received File

Download and save any file you receive in a Liberdus chat such as images, videos, music, screenshots, or documents so you can view or store it on your device.

### Steps: Saving or Downloading a File

1. Open the **chat** containing the file you want to save.  
2. Tap on the **received file preview** (image, video, audio, or document).  
3. A **download** or **file viewer** will appear.  
4. Choose **Save**, **Download**, or your system’s equivalent option.  
5. Select a folder or location on your device (**Downloads**, **Pictures**, **Documents**, etc.).  
6. Confirm the save.

The file will be saved to your device and can be opened anytime outside of Liberdus.

### Notes & Behaviors

| Rule / Behavior | Explanation |
|------------------|--------------|
| **File stays in the chat** | Saving it does not remove it from the conversation. |
| **Full quality download** | Files are not compressed- you receive the original. |
| **Open after saving** | Most devices allow opening the file directly from the download popup. |
| **Works for all file types** | Supported: images, videos, audio, and documents. |

### Important

- If you can’t find the file afterward, check your **Downloads folder** or your system’s **default save location**.  
- On mobile devices, files may save into:  
  - **Photos / Gallery** (for images)  
  - **Files / Downloads** (for documents or other media)  
- If your device asks for **storage permission**, tap **Allow** to complete the download.
