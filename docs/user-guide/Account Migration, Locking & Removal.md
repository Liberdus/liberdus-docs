## üîê How to Lock Liberdus

**Purpose:**  
Add a lock screen that requires a password for users to authenticate before accessing Liberdus or signing into any accounts.

### Steps
1. Go to **Menu**.  
2. Click **Lock**.  
3. Input your password in the **Password** box.  
4. Re-enter it in the **Confirm Password** box.  
5. Click **Save Password**.  
6. Wait a few seconds ‚Äî a notification will appear saying **Password Updated**.  

**Result:**  
Liberdus will now require your password whenever it‚Äôs opened.

### Notes
- Locking is useful for **privacy in shared environments** or on public devices.  
- Make sure your password is memorable ‚Äî there is no password recovery system.

## ‚ùå How to Remove an Account

**Purpose:**  
Delete one or more Liberdus accounts stored on your current device. This can help declutter your account list or reset your local setup.

### Steps
1. Tap the **Menu** button on the main Liberdus screen.  
2. Select **Remove** from the list of menu options.  
3. You will see the **Remove Accounts** page with two buttons:  
   - **Remove Selected**  
   - **Remove All Accounts**  
4. To remove specific accounts, tap **Remove Selected**.  
5. You‚Äôll be shown a list of your accounts.  
6. Select the ones you want to delete.  
7. Confirm when prompted to proceed.  
8. To delete every account stored on this device, tap **Remove All Accounts**.  
9. Liberdus will ask for confirmation before removing everything.  
10. Once confirmed, all accounts will be **erased from the local device**.

### Important Notes
- Removing accounts does **not delete them from the network**, only from your local device.  
- Always **back up before removing** any account ‚Äî once deleted locally, recovery is only possible using a valid backup file.  
- If you have multiple accounts, **double-check** which one you are deleting before confirming.
